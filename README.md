# 🤖 AI Job Recommender System
### 🎯 Find Your Dream Job with AI-Powered Matching

An intelligent job recommendation system that analyzes your **skills, experience, and interests** to suggest the most relevant jobs with high accuracy.

---

## ✨ Features

- 🎯 **AI-Powered Matching** – Goes beyond keyword search  
- 🧠 **Smart Recommendations** – Context-aware suggestions  
- 📊 **Real-time Match Score** – Instant results  
- ⚡ **Fast Performance** – < 0.5s response time  
- 💫 **Modern UI** – Clean & responsive design  
- 📱 **Cross-Device Support** – Works on mobile & desktop  

---

## 🧠 Tech Stack
Layer	Technology                        
🎨 Frontend	HTML, CSS, JavaScript          
⚙️ Backend	Flask (Python)          
🧠 AI/ML	scikit-learn, pandas          
💾 Database	CSV (pandas)         
## 🎬 System Workflow           
INPUT → PROCESS → OUTPUT

Skills / Resume
      ↓
TF-IDF Vectorization
      ↓
Cosine Similarity
      ↓
🎯 Best Job Matches
